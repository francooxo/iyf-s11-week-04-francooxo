## Project Description
Core JavaScript fundamentals practice: variables, data types, functions, control flow,
arrays, and objects — implemented through Lesson 7 & 8 exercises plus two projects,
an Interactive Calculator and a Student Grade Tracker.

## Technologies Used
- HTML5
- JavaScript (ES6+)

## Features
- Interactive Calculator supporting +, -, *, /, %, **
- Student Grade Tracker (add students, averages, top performer, struggling students, report cards)
- All Lesson 7 & 8 exercises

## How to Run
1. Clone this repository
2. Open any `index.html` in your browser and check the console (F12),
   or open the relevant `.js` file and run with `node filename.js`

## Lessons Learned
Practiced core JavaScript syntax and built two small applications from scratch, moving
from basic variable declarations to working with arrays of objects and object methods.

## Challenges Faced
